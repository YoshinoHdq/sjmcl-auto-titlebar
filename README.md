## 简介

Auto Titlebar 是一款 SJMCL 扩展，当鼠标悬停在启动器窗口顶部时自动显示标题栏，移出窗口时自动隐藏，为启动器提供更简洁的全屏体验。

## 功能特色

- 🖱️ 鼠标悬停时显示标题栏
- 👆 移出窗口区域时自动隐藏
- ⚙️ 可在扩展设置页开关功能
- 📦 轻量化设计，无额外依赖

## 使用方法

1. 在 Releases 下载 org.yoshino.auto_titlebar.sjmclx
2. 打开 SJMCL → 设置 → 扩展管理 → 导入
3. 启用后鼠标悬停窗口顶部即可显示标题栏

## 兼容性

- 最低启动器版本: 1.1.3
